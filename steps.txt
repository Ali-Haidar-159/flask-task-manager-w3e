
1. Create the virtual environment using these commands : 
    i. python3 -m venv venv
    ii. source venv/bin/activate

2. start the PostgreSQL in Docker  : 
    i.  docker start postgres-db
    ii. docker ps (check the container is running or not)
    iii. docker exec -it postgres-db psql -U postgres -d pythondb (access the pythondb database)
    iv. SELECT * FROM all_task; (to show all the data or task)

3. Run the init_db file to create and setup the database PostgreSQL(using Docker)

4. Check the database and table : 
    i. docker exec -it postgres-db psql -U postgres -d pythondb
    ii. docker ps
    iii. psql -U postgres -d pythondb
    iv. select * from alltask

5. Run the server : flask run 

6. Check in browser : 
    i. http://127.0.0.1:5000 ----> Home Page 
    ii. http://127.0.0.1:5000/api/tasks -----> Show all tasks
    iii. http://127.0.0.1:5000/api/tasks/<id> -----> Get a specific task
